# 二人禅 Dyad Meditation

一个用于学习和实践 Dyad Meditation（二人禅）的中文静态网站。

## 内容范围

- 二人禅的基本定义与练习结构
- 二人禅的历史起源、演变发展与当代应用
- 四轮交替练习计时器
- 新手可用探问题库
- 带领者学习路径、带领原则与注意事项
- Q&A 常见问题
- 安全边界、整合时间与带领提醒
- 基于用户提供的两本 PDF 的中文整理说明

本站内容是学习整理和实践指南，不是原书逐字翻译，也不替代心理治疗、医疗照护或合格带领者训练。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
