# 二人禅 Dyad Meditation

一个用于学习和实践 Dyad Meditation（二人禅）的中文静态网站。

## 内容范围

- 二人禅的基本定义与练习结构
- 第一次练习步骤、练习前协议和示范对话
- 四轮交替练习计时器
- 分级探问题库与练习后整合问题
- 系统学习路径与常见练习形式
- 密码访问的专业训练模块
- 密码访问的带领者模块
- Q&A 常见问题
- 适合/不适合状态、安全边界与起源发展

本站内容是学习整理和实践指南，不是原书逐字翻译，也不替代心理治疗、医疗照护或合格带领者训练。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
